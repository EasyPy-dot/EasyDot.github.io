因為官網對於E01 API的安裝說明並不是很明確, 故這裡我們補充說明並提供相對應的批次檔協助安裝.
1.先下載安裝富邦E01.
2.使用系統管理員權限開啟CMD.
3.指令: cd /d e01api根目錄.(如C:\Fubon e01\API3)
4.指令: InstallFubonE01API2.bat.

這裡我們為了簡化使用者的安裝流程, 提供一個Batch檔, 若用戶安裝E01時使用原設定值進行, 那麼在個人電腦中會將E01安裝至產相對應的位置(C:\Fubon e01), 用戶可以直接下載Batch檔執行; 若用戶將E01安裝於其它位置, 也只要將Batch檔用文字檔開啟後修正至對應的位置後存檔即可執行.

Description

因為系統架構原故, EasyPy後端仍是介接原e01 API(未來另有規劃!). 為了讓用戶專注於交易策略開發, 故我們在EasyPy與e01 API間建立管理介面, 處了進行基本的委託控管(價量), 也方便使用者進行部份管理與制定式的演算法交易. 而做為自建式的程式交易組件, 我們除了提供市場行情(縱斷面)外, 也提供部份總經數據及市場數據, 讓用戶開發橫斷面交易策略.

@Q Team是一個專注於程式、演算法及高頻交易的服務團隊, 我們的成員包含各類EA交易系統(STS, XQ, MultiCharts, E, R語言及Python)講師，專業且親切的客服同仁，以及專注於系統開發的經理人及工程師. 另外, 有別於其他團隊, 我們還加入了策略評量師與風險管理師等來服務廣大的程式交易客群.
